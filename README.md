# BloggyCms
Легкая и гибкая CMS для создания блогов и небольших сайтов с модульной архитектурой, поддержкой пользовательских полей, блочным редактором контента и системой достижений.
# Особенности

### Архитектура

+ MVC-подобная структура с разделением на контроллеры, модели и представления
+ Модульная система действий (Actions) для каждого контроллера
+ Поддержка плагинов для расширения функциональности
+ Событийная система для хуков и триггеров

### Управление контентом

+ **Посты и страницы** с категориями, тегами и архивами
+ **Блочный редактор (PostBlocks)** для гибкого построения контента
+ **HTML-блоки** для вставки готового кода
+ **Пользовательские поля** (Custom Fields) для любых сущностей
  + Текст, текстовая область, число, дата
  + Выпадающие списки, мультивыбор, чекбоксы
  + Изображения, цвет, ссылки, иконки
  + Повторители (Repeater) для повторяющихся групп

### Пользователи и права

+ **Группы пользователей** с гибкой системой прав
+ **Профили пользователей** с любым набором кастомных полей
+ **Система достижений** (Achievements) с автоматической разблокировкой
+ **Модерация комментариев** с правами на редактирование/удаление

### Шаблоны и темы

+ **Поддержка** нескольких тем оформления
+ **Кастомные шаблоны** для блоков в папке темы
+ **Asset-менеджер** для подключения CSS/JS файлов
+ **Шорткоды** для вставки динамического контента
  
### Требования

+ PHP 7.4 или выше
+ MySQL 5.7 или выше
+ Apache/Nginx с mod_rewrite
